Prepare

Clone Repository

git clone https://github.com/importantimport/hatsu.git && cd hatsu

Contributing

Go to the hatsu folder and you can see these:

  • docs - The documentation you're looking at right now, uses mdBook to build.
  • migration - SeaORM Migration.
